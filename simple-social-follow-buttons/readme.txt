== Short Description ==

Adds social media follow buttons with follower counts and allows for horizontal or vertical display.

== Description ==

Simple Social Follow Buttons plugin adds social media follow buttons with follower counts to your WordPress site. It supports Twitter, Facebook, Instagram, and YouTube. You can choose to display the buttons horizontally or vertically.

== Installation ==

1. Upload the `simple-social-follow-buttons` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings -> Social Follow Buttons to configure the plugin.

== Frequently Asked Questions ==

= How do I configure the social media accounts? =

Go to the plugin settings page under Settings -> Social Follow Buttons and enter your social media usernames or URLs.

= Can I display the buttons vertically? =

Yes, you can select either horizontal or vertical display in the plugin settings.

= How do I display the follow buttons on my site? =

You can display the follow buttons by using the shortcode `[social_follow_buttons]` in your posts, pages, or widgets.
